---CONCEITOS INICIAIS SOBRE ESTRUTURA DE DADOS, ARRAYS E REGISTRO---

**Estrutura de dados
  É uma estrutura de dados organizados na memória do computador para que os dados possam ser utilizados corretamente.

**Registro
  Permite armazenar vários tipos de variáveis. Toda estrutura de registro possui um nome, por exemplo cliente, para acessar um dado dentro deste registro deve usar “.”.Exemplo: cliente.CPF.

ENTENDA O QUE SÃO LISTAS, FILAS E PILHAS
**Listas
  As listas possuem um tamanho ajustável. Existem dois tipos de listas:
  -->Ligadas
     Existem nós onde cada um conhece o valor que está armazenado em seu interior e o valor do elemento posterior.
  -->Duplamente ligadas
     Diferente das listas ligadas, os nós dessa lista também sabem quem é o elemento anterior, além do posterior.

**Pilha ou Stack
  É uma estrutura de dados que permite o acesso a somente um dado armazenado por vez, ou seja, só pode ser lido ou removido um de cada vez. Existem dois tipos de pilhas.
  -->LIFO ou UEPS 
     UEPS (último que entra primeiro que sai) quer dizer que o último elemento inserido vai ser o primeiro a ser excluído ou manipulado.
  -->FIFO ou PEPS:
     PEPS(primeiro que entra primeiro que sai) o primeiro elemento a ser excluído ou manipulado é o primeiro.

**Filas
  O elemento removido ou manipulado é o que está na fila há mais tempo, ou seja, o primeiro, ou seja, seguindo o conceito FIFO.

ESTRUTURAS DE DADOS DO TIPO ÁRVORE, TABELA HASH E GRAFOS
**Árvores
  É um tipo de estrutura de dados que organiza os seus elementos de forma hierárquica, onde existe uma raiz que fica no topo da árvore e os elementos subordinados à raiz que fica embaixo, chamados de folhas ou nós.

**Tabela hash ou tabela de espalhamento
  Espalhar os elementos de um array(usando o hashing), de forma que fiquem não ordenados dentro de uma tabela. Para encontrar um valor utilizamos uma chave para cada valor ao invés da posição.

**Grafo
  É uma estrutura de dados não linear e permite programar a relação entre objetos. Os objetos são os vértices ou nós do grafo. E os relacionamentos são as arestas.
